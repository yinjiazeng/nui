弹出层

使用模块

下载 弹出层组件作为依赖项引入到模块中，并创建实例。

选项参数

# content

类型：{String, jQuery Object}

默认值：''

说明：弹出层内容，可以是标签字符串，也可以是jQuery对象

# template

类型：{String, Object}

默认值：''

说明：弹出层内容模版，优先级高于content，当是Object类型时，必须包含main属性作为主模版

# data

类型：{Object}

默认值：{}

说明：弹出层模版数据，用于渲染模版

# events

类型：{Object}

默认值：null

说明：用于绑定事件，用法参考代理事件

# className

类型：{String}

默认值：''

说明：添加一个或一组类名

# width

类型：{String, Number}

默认值：320

说明：宽度，可以设置为auto或者100%

# height

类型：{String, Number}

默认值：'auto'

说明：高度，可以设置为auto或者100%

# maxWidth

类型：{Number}

默认值：0

说明：最大宽度

# maxHeight

类型：{Number}

默认值：0

说明：最大高度

# zIndex

类型：{Number}

默认值：null

说明：弹出层层级

# id

类型：{String}

默认值：''

说明：标识符名称，弹出层最外层元素会增加layer-标识符名称的类名，

# skin

类型：{String}

默认值：''

说明：皮肤名称，弹出层最外层元素会增加nui-layer-皮肤名称的类名

# className

类型：{String}

默认值：''

说明：最外层元素增加类，多个类空格分隔

# timer

类型：{Number}

默认值：0

说明：自动关闭倒计时毫秒时间

# edge

类型：{Number}

默认值：0

说明：距离容器四周边缘距离

# container

类型：{Selector}

默认值：'body'

说明：显示容器

# title

类型：{String}

默认值：'温馨提示'

说明：头部标题，隐藏标题设置为null

# isMove

类型：{Boolean}

默认值：false

说明：是否可以拖动

# isInnerMove

类型：{Boolean}

默认值：false

说明：是否只能在不超过容器边缘的区域内拖动

# isMask

类型：{Boolean}

默认值：true

说明：是否显示背景遮罩

# isClickMask

类型：{Boolean}

默认值：false

说明：是否点击背景遮罩关闭弹出层

# isMoveMask

类型：{Boolean}

默认值：false

说明：是否在拖动弹出层时，显示一个与弹出层同等大小的背景遮罩随鼠标移动

# isHide

类型：{Boolean}

默认值：false

说明：是否能使用hide方法关闭

# isCenter

类型：{Boolean}

默认值：true

说明：是否垂直水平居中显示

# isFull

类型：{Boolean}

默认值：false

说明：是否全屏显示

# isTop

类型：{Boolean}

默认值：false

说明：是否置顶显示

# isTips

类型：{Boolean}

默认值：false

说明：是否以贴士形式展示，无标题与底部按钮

# isFixed

类型：{Boolean}

默认值：true

说明：是否拖动容器滚动条时弹出层固定所在位置

# scrollbar

类型：{Boolean}

默认值：true

说明：当内容超出时，是否显示滚动条，如果设置为false并且内容区域超出了maxHeight，也会有滚动条

# align

类型：{String}

默认值：'center'

说明：底部按钮对齐方式，left / center / right

# bubble

类型：{Object}

属性：enable {Boolean} false 是否启用

属性：dir {String} 'top' 箭头方向 top / right / bottom / left

说明：是否以气泡形式显示，弹出层边缘会多出箭头

# iframe

类型：{Object}

属性：enable {Boolean} false 是否启用

属性：cache {Boolean} false 是否缓存页面

属性：src {String} '' 请求地址

说明：加载iframe内容，不建议跨域使用

# close

类型：{Object}

属性：enable {Boolean} true 是否启用

属性：text {String} 'X' 按钮内容

说明：头部关闭按钮

# confirm

类型：{Object}

属性：enable {Boolean} false 是否启用

属性：text {String} '确定' 按钮内容

属性：name {String} 'normal' 设置按钮类型名称，具体参考按钮元素

属性：callback {Function} function(){return true} 触发回调，返回值为true时才能关闭弹出层

说明：底部确认按钮

# cancel

类型：{Object}

属性：enable {Boolean} true 是否启用

属性：text {String} '取消' 按钮内容

属性：callback {Function} undefined 触发回调，若返回值是false，则无法关闭弹出层

说明：底部取消按钮

# button

类型：{Array Object}

默认值：null

属性：id {String} 按钮id，若取名为cancel / confirm / cancel会覆盖内置这三个按钮的配置属性

属性：text {String} 按钮内容

属性：name {String} 设置按钮类型名称，当值设置为disabled时，表示按钮被禁用，无法触发回调，具体参考按钮元素

属性：style {Object} 设置按钮行内样式

属性：callback {Function} 触发回调，若返回值是false，则无法关闭弹出层，若id是confirm，返回值必须是true才能关闭

说明：设置底部自定义按钮

# position

类型：{Object}

属性：top {Number, String} 0 距离容器顶部距离

属性：right {Number, String} 0 距离容器右侧距离

属性：bottom {Number, String} 0 距离容器底部距离

属性：left {Number, String} 0 距离容器左侧距离

说明：弹出层定位，top/bottom或者left/right不可同时设置，只能设置top/left、top/right...以此类推。属性值中可使用运算符计算距离，其中self表示定位方向弹窗大小，例如top中的self表示弹出层自身高度，left中的self表示弹出层自身宽度。

# under

类型：{Array, Layer Object}

默认值：null

说明：将一个或者一组弹出层置于背景遮罩底部，参数值只能是layer实例或者layer实例的集合，在当前弹出层关闭时，被置于底部的弹出层会恢复原来的层级状态。

# onInit

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：弹出层初始化时回调函数。

# onHideBefore

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：弹出层调用hide方法被销毁之前回调，若返回false，则不能销毁该弹出层。

# onDestroy

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：弹出层关闭销毁时回调函数。

# onDestroyBefore

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：弹出层在销毁之前回调，若返回false，则不能销毁该弹出层。

# onMove

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：拖动弹出层重新定位时回调函数，isMove为true时才会触发。

# onResize

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：窗口改变大小，使弹出层位置发生变化时回调，在设置position或者isCenter为true时才会触发。

# onReset

类型：{Function}

参数：self {Layer Object} 当前layer实例

说明：重置实例为初始化状态时回调。

# onScroll

类型：{Function}

参数：self {Object} 当前layer实例

参数：event {Event Object} 事件对象

参数：elem {jQuery Object} 滚动元素jQuery对象

参数：data {Object} 滚动数据，包含top以及left

说明：容器滚动时触发回调。

# onTimer

类型：{Function}

参数：self {Layer Object} 当前layer实例

参数：time {Number} 定时毫秒数

说明：定时关闭弹出时触发回调，选项参数timer大于0时才会启用。

实例属性与方法

# element

类型：{jQuery Object}

说明：对应类名为nui-layer元素

# head

类型：{jQuery Object}

说明：对应类名为layer-head元素

# main

类型：{jQuery Object}

说明：对应类名为layer-main元素

# foot

类型：{jQuery Object}

说明：对应类名为layer-foot元素

# data

类型：{Object}

属性：width {Number} 自身宽度，不包含边框边距

属性：height {Number} 自身高度，不包含边框边距

属性：outerWidth {Number} 自身完整宽度

属性：outerHeight {Number} 自身完整高度

属性：top {Number} 距离容器顶部距离，包括垂直滚动距离

属性：left {Number} 距离容器左侧距离，包括水平滚动距离

属性：offsetTop {Number} 距离容器顶部距离

属性：offsetLeft {Number} 距离容器左侧距离

说明：弹出层自身属性

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：设置选项参数

# reset

类型：{Function}

说明：将实例还原为初始状态

# resize

类型：{Function}

说明：改变实例大小位置为正常状态

# hide

类型：{Function}

说明：销毁实例，当实例选项参数中的isHide为false，则无法销毁

# destroy

类型：{Function}

说明：销毁实例

静态方法

# config

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：全局设置默认选项参数，需在调用组件前设置方可生效

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

参数：id {Number, String} 实例id或者选项参数id

说明：设置选项参数，不传id则对所有实例操作

# reset

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：将实例还原为初始状态，不传id则对所有实例操作

# resize

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：改变实例大小位置为正常状态，不传id则对所有实例操作

# hide

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：销毁实例，当实例选项参数中的isHide为false，则无法销毁，不传id则对所有实例操作

# destroy

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：销毁实例，不传id则对所有实例操作
